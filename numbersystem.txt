decimal number system (0 to 9) 10 digits

56 = 50 + 6 = 5*10^1 + 6*10^0
123 = 100 + 20 + 3 = 1*10^2 + 2*10^1 + 3*10^0

binary number system (0,1) 2 digits

101 = 1*2^2 + 0*2^1 + 1*2^0 = 4+0+1 = 5

111 = 1*2^0 + 1*2^1 + 1*2^2 =7

hexadecimal number system (0 to 9, A-F) 16 digits
10-A 11-B 12-C 13-D 14-E 15-F

AD = D*16^0 + A*16^1 = 13*1 + 10*16 = 13+160 = 173

255 = 255/16 = 1515 = FF
